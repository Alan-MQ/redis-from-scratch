# Redis From Scratch - Project Context

## 项目愿景
构建一个功能完整的迷你Redis实现，重点在于深入理解Redis的核心机制和分布式系统概念。

## 学习目标
- **数据结构掌握**: 深入理解Redis各种数据类型的实现原理
- **网络编程**: 掌握高性能网络服务器设计
- **持久化机制**: 理解RDB和AOF的实现细节
- **分布式系统**: 学习集群、主从复制、分片等分布式概念
- **内存管理**: 理解Redis的内存优化策略

## 技术选型
- **语言**: Go (语法简洁，并发模型优秀，开发效率高)
- **网络**: 原生net包 + goroutine
- **序列化**: 自实现RESP协议
- **测试**: Go内置testing包 + testify断言库

## 项目结构
```
redis-from-scratch/
├── .claude/              # Claude AI助手上下文
├── doc/                  # 学习文档和记录
├── src/
│   ├── core/            # 核心数据结构和命令处理
│   ├── network/         # 网络层和协议实现
│   ├── storage/         # 持久化和存储
│   ├── cluster/         # 集群和分布式功能
│   └── client/          # 客户端实现
├── tests/               # 测试用例
└── examples/            # 使用示例
```

## 学习方式
1. 每个功能都会有详细的设计文档
2. 关键代码由学习者实现，导师提供指导和框架
3. 每个阶段都有明确的学习目标和测试验证
